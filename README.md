# 📐 Class Diagram Designer

An interactive, browser-based UML class diagram editor. Build, model, and visualize class diagram (software architectures) with a zoomable canvas, live property inspector, and local persistence.

---

## ✨ Key Features

* **Interactive Canvas:** Drag and snap entities to a visual grid with high-performance pan and zoom.
* **Smart Connectors:** Draw UML relationships from anchor-to-anchor with live preview lines and Manhattan routing.
* **Live Inspector:** Real-time editing of class members (attributes/operations) and relationship properties.
* **Quick-Switch Hotkeys:** Change relationship types on the fly using keys **1-6** during connection.
* **Auto-Persistence:** Your progress is automatically saved to `localStorage`, no login required.

---

## 🚀 Quick Start

### 1. Install dependencies

```bash
npm install

```

### 2. Run development server

```bash
npm run dev

```

### 3. Build & Preview

```bash
npm run build
npm run preview

```
